# AWS Cloud Automation (Mobile-First Lab)
Developed an industry-level cloud automation tool using **Golang** on **Android/Termux**.

## Tech Stack
* **Language:** Go
* **SDK:** AWS SDK v2
* **Environment:** Termux (Linux on Mobile)

## Key Features
* **Custom S3 Mock:** Created a mock server to test cloud operations locally.
* **Optimized Build:** Used `CGO_ENABLED=0` for portable compilation.
